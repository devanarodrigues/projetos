Detective Comics

  - Primeiro projeto após iniciar nos estudos de Web Development.

  - O projeto possui duas versões: a primeira foi onde comecei a aplicar tudo que havia aprendido até o momento, já a segunda foi onde eu aperfeiçoei o design do site e o deixei responsivo após ter um certo domínio do assunto.

  - Justamente por ser o primeiro projeto que realizei, tive bastante dificuldade em posicionar os elementos, o que resultou em designs indesejados, mas que foram corrigidos na segunda versão do projeto.

  - As tecnologias utilizadas nele foram apenas HTML5 e CSS3. A bibilioteca de icones FontAwesome também foi manuseada.

Portifólio pessoal

  - Segundo projeto após iniciar nos estudos de Web Development.

  - Este projeto foi criado com o intuito de me apresentar para o usuário e também os meus conhecimento.

  - Durante o desenvolvimento não foram encontradas dificuldades para posicionar os elementos utilizando Flexbox, ao contrário do primeiro projeto.

  - Foi o projeto onde tive o primeiro contato com responsividade. Existiram certas dificuldades para fazer a adaptação.

  - As tecnologias utilizadas nele foram apenas HTML5, CSS3 e JavaScript. As bibiliotecas FontAwesome e ScrollRevealJS também foram manuseadas.
